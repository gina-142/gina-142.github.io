The Alcuiberre Equation

Alcuiberre attempted, in his 1994 publication, to find a mathematical way to express the warp drive. Theoretically, this could show how a warp drive could be created (in reality, there are a few problems with this, see the next page 'Fact or Fiction'). The details behind his derivation use elements of General Relativity, but it is possible to explore the ideas Alcuiberre introduced by looking at the following function:

$f(r_s) = \frac{tanh(\sigma (r_s + R)) - tanh(\sigma (r_s - R))}{2 \cdot tanh(\sigma R)}$

This can be thought of a shape function to represent the shape of the warp bubble, rather than representing any physical quantities. Alcuiberre developed a series of metrics, which describe the properties of the warp drive, such as the spacial geometry and velocity components. This function $f(r_s)$ is used in the metric that describes 'the velocity profile of the warp bubble along the x-axis'. In simpler terms, this shows how spacetime moves relative to an outside observer. So if we were to look at a warp bubble, this function describes the geometry of what we would see.

The top hat shape function can be observed by plotting this on a graph. The function outputs a value between 0 and 1. Physically, when it is 0, it means that there is no change to space time, i.e. outside the bubble and at 1, which occurs inside the bubble, the full effect of the warp is experienced, the ship being carried along with the bubble. The values inbetween represent the bubble wall, where the transition from normal to shifted spacetime occurs. Changing the \sigma parameter changes how quickly this whift occurs, and this also has an effect on the energy required for the bubble (see 'Fact or Fiction').
